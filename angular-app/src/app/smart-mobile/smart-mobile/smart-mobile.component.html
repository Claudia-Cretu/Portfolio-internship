Smart Mobile Projects

I am currently in semester four, Smart mobile. In this semester, I am learning how to create mobile applications using the Flutter framework and Kotlin.

Flutter Projects

Here are some of the projects I have worked on during this semester:

  • Project 1 - "Throw your phone game app": This app is a game where two users throw their phones, and the device calculates who threw it higher to determine the winner. I created this app using Flutter and developed it in Android Studio. To store game data and user information, I connected the app to a MongoDB database. Key technical aspects included:
    • Implementing game logic to calculate the height of the phone throw using device sensors.
    • Developing a calibration feature to improve height accuracy.
    • Creating a login and registration system to manage user accounts and game history.
    • Designing a game screen with interactive elements like a "Play again" and "Back to login" button.
    My takeaways from this project include learning how to create games with Flutter, programming in Dart, and using Android Studio for development.
  • Project 2 - "Talk with strangers app": This app connects pet owners living nearby who share similar interests, facilitating social connections through a chat feature. I worked on the conversation list page, implementing a search bar with a TextEditingController to filter contacts based on their names. Key technical aspects included:
    • Designing a user interface for the conversation list page.
    • Implementing a search function that dynamically updates the contact list as users type.
    • Practicing Dart syntax and exploring Flutter's capabilities for building interactive UI components.
    This project helped me further develop my skills in Dart and Flutter, particularly in creating interactive UI elements.
